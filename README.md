# ML Classification Demo

A minimal machine learning project demonstrating a full pipeline from data loading to evaluation using scikit-learn.

## Features
- Data loading from sklearn dataset
- Train/test split
- Feature scaling
- Logistic regression model
- Evaluation metrics
- Model saving with joblib

## Project Structure
See repository tree for details.

## How to run

```bash
pip install -r requirements.txt
python main.py
````

## Model

Logistic Regression (binary classification)

## Output

* Accuracy score
* Classification report
